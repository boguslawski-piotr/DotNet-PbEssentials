Class EncryptedFileSystem

Decorator for classes that implement the IFileSystem interface which provides encryption of stored files and of course decrypting when reading.

Id passed to constructor will be used as directory name in the root of file system. The data in this directory (and in all subdirectories) will be encrypted. The data in another directories (unless the Id was set to null or empty string) will not be encrypted.
