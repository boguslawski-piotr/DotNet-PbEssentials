Class Settings 

public class Settings : Observable, IEnumerable<KeyValuePair<string, object>>

Class that makes it easy to handle all sorts of settings, more precisely, values of any type that is accessed through a key.

The class is fully ready for use in binding systems because it inherits after Observable.

The class can also be used as a regular collection, that is, it can be enumerated :)

Examples

public class Main
{
    public static void Main(string[] args)
    {
        MySettings s = new MySettings();
        s.Load();

        Console.WriteLine($"{s.BoolSetting}, {s.StringSetting}");
        Console.WriteLine($"---{s.Get<string>("test")}---");

        s["test"] = "Hello!";
        s.BoolSetting = false;
        s.StringSetting = "another value";

        // Assuming that the data has been saved, that is you implemented 
        // virtual functions LoadAsync and SaveAsync or
        // used SettingsInStorage as base class for MySettings.
        //
        // The first time you start the program, you should see:
        //  
        // True, some value
        // ------
        //
        // The next time you run the program, you should see:
        //
        // False, another value
        // ---Hello!---
    }
}

public class MySettings : Settings
{
    [Default(true)]
    public bool BoolSetting
    {
        get => Get<bool>();
        set => Set(value);
    }

    [Default("some value")]
    public string StringSetting
    {
        get => Get<string>();
        set => Set(value);
    }

    }

    {
        // Save your settings as you like.

        // Or use as a base class one of the predefined classes

    }
}

GetDefault(String)

protected virtual object GetDefault(string key)

Gets default value, specified in Settings.DefaultAttribute attribute, for a property whose name matches the content of key.

The function can be overrided and used to provide more complex default values that can not be passed with the Settings.DefaultAttribute attribute.

GetEnumerator()

public IEnumerator<KeyValuePair<string, object>> GetEnumerator()

Returns an enumerator that iterates through the entire settings collection.

Remarks

The enumerator is safe to use concurrently with reads and writes to the settings collection, however it does not represent a moment-in-time snapshot. The contents exposed through the enumerator may contain modifications made to the settings collection after GetEnumerator was called.
